【屬性：STR (力量 / Strength)】
關鍵字：肌肉, 負重, 搬運, 徒手傷害, 力量檢定
描述：代表調查員的肌肉力度。
功能：
1. 判定搬運、推拉重物、擒抱的能力。
2. 決定近戰攻擊的「傷害獎勵 (Damage Bonus)」。
後果：若 STR 減至 0，調查員將殘廢無法行動。

【屬性：CON (體質 / Constitution)】
關鍵字：健康, 抵抗毒素, 疾病, 生命力, 憋氣, 抵抗力
描述：代表身體素質、活力與精力。
功能：
1. 抵抗毒藥、疾病、溺水、窒息。
2. 決定「生命值 (HP)」的主要屬性之一。
後果：若 CON 減至 0，調查員死亡。

【屬性：SIZ (體型 / Size)】
關鍵字：身高, 體重, 體格, 穿過狹縫, 視野
描述：衡量人物的高矮胖瘦。
功能：
1. 決定是否能越過障礙物看東西，或擠過狹小通道。
2. 決定「生命值 (HP)」與「傷害獎勵 (Damage Bonus)」。
後果：若 SIZ 減至 0，代表該角色消失。

【屬性：INT (智力 / Intelligence)】
關鍵字：學習, 記憶, 分析, 邏輯, 理解
描述：代表學習能力、記憶力與分析力。
衍生檢定：
1. **靈感 (Idea Roll)**：INT x 5。用於理解事物、連結線索。
2. **興趣技能點**：由 INT 決定。
3. **法術學習**：影響學習克蘇魯咒語的速度。
後果：若 INT 為 0，角色無智力可言。

【屬性：POW (意志 / Power)】
關鍵字：意志力, 精神力, 魔法值, 幸運, 精神對抗
描述：代表意志力與精神強度。
衍生檢定：
1. **幸運 (Luck Roll)**：POW x 5。
2. **心智 (Sanity)**：起始值 = POW x 5。
3. **魔法值 (Magic Points)**：上限等於 POW。
特性：除非特殊情況，POW 的損失通常是永久的。成功使用魔法可能增加 POW。

【屬性：DEX (敏捷 / Dexterity)】
關鍵字：速度, 靈巧, 平衡, 閃避, 先攻
描述：代表速度、靈巧度與身體柔軟度。
功能：
1. 決定戰鬥中的行動順序 (DEX 越高越先動)。
2. 判定抓住落物、保持平衡、靈巧動作。
3. **閃避 (Dodge)**：起始值 = DEX x 2。
後果：若 DEX 為 0，角色極度不協調，無法行動。

【屬性：APP (外表 / Appearance)】
關鍵字：長相, 魅力, 第一印象, 社交
描述：代表外表的吸引力與友好度。
功能：影響社交場合的第一印象、快速交談 (Fast Talk) 與議價 (Bargain)。
限制：APP 只代表外貌，不等於領導力（領導力屬扮演範疇）。

【屬性：EDU (教育 / Education)】
關鍵字：知識, 學歷, 學校, 母語
描述：代表通過學校學習的知識水平。
衍生檢定：
1. **知識 (Know Roll)**：EDU x 5。
2. **職業技能點**：由 EDU 決定。
3. **母語 (Own Language)**：起始值 = EDU x 5。
參考：EDU 12 = 高中畢業；EDU 16 = 大學學位。

【衍生屬性：SAN (心智 / Sanity)】
關鍵字：理智, 發瘋, 精神崩潰, 恐懼
定義：
1. **心智 (Sanity)**：起始值 = POW x 5。
2. **心智點 (Sanity Points)**：當前剩餘的理智，會隨遊戲變動。
3. **最大心智值 (Max Sanity)**：99 - 克蘇魯神話技能 (Cthulhu Mythos)。
規則：
1. 遭遇怪物、超自然現象或施法會扣除心智點。
2. 單次扣除過多可能導致瘋狂。
3. 心智點可緩慢恢復，但極難。

【衍生檢定：靈感 (Idea Roll)】
公式：INT x 5
時機：當沒有合適技能時使用。
用途：
1. 調查員是否能觀察或理解所見事物。
2. 是否感受到特殊召喚或直覺。
3. 連結線索與情報。

【衍生檢定：幸運 (Luck Roll)】
公式：POW x 5
時機：判定運氣相關的事件。
用途：
1. 是否剛好帶了某裝備。
2. 怪物隨機攻擊誰。
3. 環境是否發生巧合（如地震、樓梯斷裂）。
4. 在緊急時刻取代跳躍或閃避檢定（視 KP 決定）。

【衍生檢定：知識 (Know Roll)】
公式：EDU x 5
時機：判定角色是否知道某項一般知識。
用途：
1. 辨認俗語、俚語。
2. 具備基礎科學常識（如化學反應、地理資訊）。
限制：知識檢定上限為 99。

【衍生屬性：傷害獎勵 (Damage Bonus / DB)】
公式：基於 (STR + SIZ) 查表決定（通常是 +1D4, +1D6 等）。
規則：
1. 近戰攻擊：加上完整 DB。
2. 投擲武器：加上 DB 的一半。
3. 火器/槍械：不加 DB。
4. 咬擊：不加 DB。

【衍生屬性：生命值 (Hit Points / HP)】
公式：(CON + SIZ) / 2 (無條件進位)。
規則：
1. **受傷**：優先扣除 HP。
2. **昏迷**：HP <= 2，角色不省人事。
3. **死亡**：HP <= 0，若當回合結束前未回復至 1 以上，角色死亡。
恢復：
1. 自然恢復：每週 1D3。
2. 緊急治療 (急救/醫學)：立即恢復 1D3 (同一傷口限一次)。

【衍生屬性：魔法值 (Magic Points / MP)】
公式：上限等於 POW。
規則：
1. **消耗**：施法或對抗邪惡影響時扣除。
2. **耗盡**：MP = 0 時，角色昏迷，直到回復至少 1 點。
3. **恢復**：自然恢復，每 24 小時回復至滿值。
